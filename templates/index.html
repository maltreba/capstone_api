Welcome To Maltreba API!

This API provides you data related to Chinook Store.

 

Static API

/playlist

Examples

https://maltreba-api.herokuapp.com/playlist

Dynamic API

/customer/<country>/<year>
/product/track/<Genre>
/sales/<year>
 
Examples
https://maltreba-api.herokuapp.com/customer/USA/2009
https://maltreba-api.herokuapp.com/product/track/Pop
https://maltreba-api.herokuapp.com/sales/2009
